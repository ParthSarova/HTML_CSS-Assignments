## 🚀 How to Use

1. **Open in Browser:**
   - Double-click any `.html` file to open it in your default web browser
   - Or right-click → "Open with" → Choose your preferred browser

2. **Edit Files:**
   - Open any `.html` file in a text editor (VS Code, Sublime, Notepad++, etc.)
   - Modify HTML content and styling
   - Save and refresh in browser to see changes

3. **Local Media Files:**
   - Keep all associated media files (`.mp4`, `.mp3`, images) in the same directory as the HTML file
   - Update file paths in `src` attributes if media is moved to a subfolder
