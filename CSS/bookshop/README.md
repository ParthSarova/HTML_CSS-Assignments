# BookNest Landing Page

This repository contains a foundational HTML landing page for a fictional bookstore named "BookNest." It serves as a basic structural skeleton, welcoming users with an inspirational quote by Ernest Hemingway and a simple call-to-action button to explore the collection. Below the introductory header, the page features a dedicated section showcasing a curated gallery of books, including titles like *The Book Thief*, *Atomic Habits*, *Becoming*, and *Educated*. Built entirely with standard HTML5, the layout currently relies on fundamental table elements to organize the book covers side-by-side. To view or modify this project, simply ensure that the referenced local image files are placed in the same directory as the HTML document, and open the file in any modern web browser.
