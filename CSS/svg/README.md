# SVG Shapes and Animations Showcase

This file serves as an interactive showcase of inline Scalable Vector Graphics (SVG) and native SMIL animations embedded directly within the HTML Document Object Model. It provides a wide variety of vector drawing examples, ranging from basic geometries like circles, rectangles, and polygons to complex organic shapes crafted using bezier curves. Beyond static drawings, the document heavily features native SVG animation tags to demonstrate dynamic effects, including horizontal movement, seamless color transitions, opacity pulsing, and line-drawing techniques utilizing stroke dash properties. A standout feature is the animated, beating heart created through a combination of custom paths and multi-color gradient animations. Because all styling, drawing, and animation logic is contained within the SVG tags themselves, this HTML5 project requires no external assets or stylesheets and can be run immediately by opening it in any standard web browser.
