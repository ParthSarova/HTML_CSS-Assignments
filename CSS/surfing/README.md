# WaveRiders Surf Academy

The WaveRiders Surf Academy project is a modern, single-page promotional website designed for a fictional surf school. It acts as a comprehensive showcase of modern CSS capabilities, with all styling housed directly within the document's `<style>` tag. The page features a dynamic hero section with a dark gradient overlay, responsive typography using the `clamp()` function, and a sticky navigation bar that employs glassmorphism and animated hover effects. Further down, the site presents interactive course cards that utilize CSS 3D transforms for engaging hover states, alongside a dedicated gallery section with grayscale image processing that reveals full color upon user interaction. Built with HTML5 and advanced CSS3 techniques like Flexbox, custom variables, and keyframe animations, this project requires no external stylesheets. To view it, simply place your chosen background and gallery images in the project directory and open the HTML file in a browser.
