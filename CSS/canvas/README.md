# Canvas Glass Bubbles & Magnification Effect

This project features an interactive HTML5 Canvas animation demonstrating a sophisticated "glass bubble" refraction effect. Built entirely with the native JavaScript Canvas API, the script renders a stylized blue gradient background with smoothly bobbing typography that commemorates a famous cricket commentary moment. The core technical highlight is the custom magnification logic, which uses canvas clipping paths (`ctx.clip()`) and scaling transformations to create a realistic lens effect, magnifying the text seamlessly as translucent bubbles float upward across the screen. This single-file project requires no external dependencies aside from Google Fonts for typography, making it an excellent, lightweight reference for advanced 2D canvas manipulation, state saving/restoring, and continuous animation loops using `requestAnimationFrame`. To view the animation, simply open the HTML document in any modern web browser.
