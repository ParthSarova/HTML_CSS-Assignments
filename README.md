# HTML & CSS Assignments

This repository contains various assignments related to HTML and CSS. Each assignment showcases different aspects of web development using these essential technologies.

## Objectives
- To understand the fundamentals of HTML5 and CSS3.
- To create responsive and accessible web pages.
- To apply CSS for styling and layout.

## Assignments Overview
- **Assignment 1**: Basic HTML structure, elements, and semantic HTML.
- **Assignment 2**: Introduction to CSS, selectors, and properties.
- **Assignment 3**: Responsive web design techniques using media queries.
- **Assignment 4**: CSS Flexbox and Grid layouts.
- **Assignment 5**: Combining HTML and CSS for a complete web page.

Feel free to explore each assignment and learn how to build beautiful web pages!
